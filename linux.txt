1. Перейдитe в папку html и выведите список всех папок(без файлов)

    cd /var/www/html/ # перехожу в папку html
    find . -maxdepth 1 -type d # вывожу список всех папок
    
    
2. Удалите пустые строки из файла error.txt

    cd /var/www/html/logs/ # перехожу в папку logs
    sed -ie "/^$/d" error.txt # удаляю пустые строки из файла error.txt
    
3. Выведите список файлов с расширением .jpg в папке images

    find /var/www/html/images -name '*.jpg' # вывожу список файлов с расширением .jpg в папкеimages
    
    
4. Напишите bash скрипт который переименует все файлы в папке logs с .txt на .log

    touch script # создаю скрипт файл
    cat > script # делаю запись в скрипт файл
    #!/bin/bash # указываю оболочку для скрипта
    cd /var/www/html/logs/ # перехожу в папку logs
    rename 's/txt/log/' *.txt # переименовываю файлы с .txt на .log 
    ^C # завершаю запись в файл
    chmod +x script # делаю файл исполняемым
    
    
5. Модифицируйте права доступа и владельца папки images следующим образом: Пользователь www-data 
и группа www-data имеют чтение/запись на всю дерикторию и на файлы внутри дериктории остальные не
имеют доступа к папке и файлам.

    sudo chown -R www-data /var/www/html/images # изменяю влядельца для папки images и файлов внутри нее
    sudo chgrp -R www-data /var/www/html/images # изменяю группу для папки images и файлов внутри нее
    sudo chmod -R ugo-rwx /var/www/html/images # забираю все права доступа на папку images и файлы внутри нее у 
                                                пользователя, группы и остальных
    sudo chmod -R  ug+rw /var/www/html/images # добавляю права на чтение и запись влядельцу и группе на 
                                                папку images и файлы внутри нее
                                                
                                                
6. Удалите все содержимое в файле access.txt не удаляя сам файл.

    cd /var/www/html/logs/ # перехожу в папку logs
    > access.txt # удаляю все содержимое в файле
    
    
7. Выведите список файлов в папке images размер которых более 2 мегабайт

    find /var/www/html/images -size +2000k # вывожу  список файлов в папке images размер которых более 2 мегабайт
